CONCEPTS DU MODELE RELATIONNEL

  • UN DOMAINE EST UN ENSEMBLE DE VALEURS CARACTERISE PAR UN NOM

    • Le domaine des nombres entiers
    • Le domaine des couleurs primaires
    • Le domaine des dates

      Remarque : Dans les SGBD relationnels on utilise plus couramment la notion de type

  • UNE TABLE EST UN SOUS-ENSEMBLE DU PRODUIT CARTESIEN DE DOMAINES CARACTERISE PAR UN NOM.
