DEMARCHE DE CONSTRUCTION DU MCD Etape 2.3: le Graphe de Dépendances fonctionnelles

  • Dépendance fonctionnelle élémentaire:

    • Définition: Dépendance fonctionnelle de la forme X -> A, où A est un attribut unique non inclus dans X (A ? X) et où il n ’existe pas X ’ ? X tel que X ’ -> A;
    • Exemple: NBON -> NCLI
    • Contrexemple: (NBON, DATE) -> NCLI
    • Propriété: Transitivité uniquement.

  • Graphe de dépendances fonctionnelles:

    • Définition: Un graphe qui permet de visualiser les dépendances fonctionnelles élémentaires est appelé graphe des dépendances fonctionnelles.
