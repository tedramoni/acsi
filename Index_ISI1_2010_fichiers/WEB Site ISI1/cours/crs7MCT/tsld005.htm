LE CONCEPT D’EVENEMENT

  • Définition: L’événement est la représentation d’un fait nouveau pour le système étudié.

    L’événement est porteur d ’information.

  • Exemples:

    • L’arrivée d’une « déclaration de sinistre » dans le système d’information d’une compagnie d’assurance est un événement porteur d’information concernant l’assuré, le bien assuré et le sinistre affectant celui-ci.
    • L’événement peut être le constat d’un non évènement, comme « l’absence de réponse à une demande ».

  • Formalisme: Un évènement est représenté

    par un cercle contenant la désignation de

    l’événement.
