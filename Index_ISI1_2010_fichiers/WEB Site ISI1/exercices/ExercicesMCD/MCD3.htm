3: La société française d'archéologie

 

La société française d'archéologie (SFA) aimerait constituer une banque de données et gérer une bibliothèque. On dispose des données suivantes pour décrire le monde réel de cette application.

Propriétés:

NUMOBJ: Numéro d'un objet archéologique

DESOBJ: Désignation de l'objet

TYPOBJ: Type d'objet

DATOBJ: Date d'origine

CODVIL: Code d'une ville

ANOMVIL: Ancien nom d'une ville

MNOMVIL: Nom moderne d'une ville

CODMUS: Code d'un musée

NOMMUS: Nom d'un musée

CODSIT: Code d'un site

DESSIT: Désignation du site

CIVSIT: Civilisation à laquelle le site est

attribué

NOOUV: Numéro d'un ouvrage

TITOUV: Titre de l'ouvrage

NUMEDI: Numéro de l'éditeur

DESEDI: Désignation de l'éditeur

DATEDI: Date d'édition

NUMAUT: Numéro d'un auteur

NOMAUT: Nom de l'auteur

On sait de plus que:

  • Un objet provient d'un site donné et est exposé dans un musée donné.
  • Un ouvrage se rapporte à un ou plusieurs sites et fait référence à plusieurs objets.
  • Un ouvrage peut avoir plusieurs coauteurs et un seul éditeur.
  • Les sites et les musées sont situés dans des villes.
