1: Le suivi des réclamations

Corrigé:

I Transformation des entités avec leurs propriétés et clés

  • CONSOMMATEUR (code consom, nom consom., prénom consom., adresse consom.)
  • USINE (code usine)
  • PRODUIT (code produit)
  • REPONSE (code réponse, date réponse, libellé réponse)
  • GROUPE DE PRODUIT (code groupe)

II Transformation des associations à cardinalités (0,1) ou (1,1)

  • RECLAMATION(code réclamation, date réclamation, code consom#., code produit#, code usine#)

Remarque: RECLAMATION résulte également de l'application de la première règle de passage: la transformation des entités en relations/tables.

III Transformation des associations ayant uniquement des cardinalités (0,n) ou (1,n)

  • REGROUPEMENT DE GROUPES (code groupe aval#, code groupe amont#)
  • APPELER (code réclamation#, code réponse#)
  • GROUPER (code groupe# , code produit#)
